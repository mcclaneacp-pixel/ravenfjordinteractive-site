About Ravenfjord Interactive

Ravenfjord Interactive is an independent game studio founded by Jeffrey Graves. The studio focuses on responsive gameplay, meaningful progression, replayability, and difficulty that rewards practice and player improvement.

Each project is developed through careful prototyping, frequent testing, and direct player feedback.
